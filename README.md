# GetMobil Project

# Requirements
* Selenium 4.6
* Cucumber 6.11
* Java 16
* Maven
* TestNG

# Installation

* Firstly clone the project
```
git clone https://github.com/busedemirbas12/GetMobilWeb
```

## For Allure Report

* Firstly run the test TestRunner class
* Then open the terminal
* Go to project path
* And run this command "allure serve"
